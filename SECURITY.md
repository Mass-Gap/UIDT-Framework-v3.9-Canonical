# Security Policy — UIDT Framework v3.9

## Supported Versions

| Version | Supported |
|---------|-----------|
| 3.9.x (Canonical) | ✅ Active |
| < 3.9 | ❌ Superseded |

## Reporting a Vulnerability

If you discover a security vulnerability in the verification scripts, CI workflows, or dependency chain, please report it responsibly:

**Contact:** [badbugs.arts@gmail.com](mailto:badbugs.arts@gmail.com)

**Scope:**
- Verification scripts (`verification/`)
- CI/CD workflows (`.github/workflows/`)
- Dependency vulnerabilities (`requirements.txt`)
- Data integrity issues in `CANONICAL/` or `LEDGER/`

**Out of Scope:**
- Theoretical physics claims (use [Issue Templates](https://github.com/Mass-Gap/UIDT-Framework-v3.9-Canonical/issues/new/choose) instead)
- Feature requests

**Response Time:** We aim to acknowledge reports within 72 hours.
