# UIDT Verification Data

This directory contains verification reports, lattice comparison data, and numerical outputs
generated by the UIDT automated verification pipeline (AVP).

## Contents

- Verification reports (per framework version)
- Lattice QCD comparison datasets
- RG fixed-point analysis outputs
- Kappa scan results
- Error propagation analyses

## Usage

Verification scripts in `verification/scripts/` write their outputs here.
All data files are read-only reference artifacts — do not modify manually.
