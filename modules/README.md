# UIDT Physics Modules

**Physical Implementations of UIDT Framework**

## Overview

Six physics modules implementing specific aspects of the UIDT framework.

## Modules

| Module | Purpose | Claims |
|--------|---------|--------|
| `geometric_operator.py` | Geometric operator construction | C-001, C-002 |
| `lattice_topology.py` | Lattice discretization | C-010, C-024 |
| `covariant_unification.py` | Covariant formulation | C-015 |
